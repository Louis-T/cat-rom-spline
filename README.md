> Catmull Rom spline interpolation made easy.


## Install

```sh
$ npm install --save cat-rom-spline
```


## Usage

```js
var catRomSpline = require('cat-rom-spline');

catRomSpline('Rainbow');
```

```sh
# creates a browser.js
$ npm run browser
```
